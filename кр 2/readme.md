
# Временная сложность 

 O(n_inter ⋅n_ants ⋅n 2 )
 
 # Оценка временной сложности:
 
 Внешний цикл (n_iter) — выполняется n_iter раз.

Цикл по муравьям (n_ants) — для каждого муравья:

Построение пути: O(n2)

На каждом из n шагов муравей просматривает до n непосещённых городов, чтобы вычислить вероятности.

Вычисление длины пути: O(n).

Обновление феромонов: O(n2)

Испарение: проход по всей матрице n×n.

Отложение: для каждого пути проходим n рёбер, но в худшем случае все муравьи могут проходить разные рёбра, поэтому считаем как O(n 2).

 Эвристические алгоритмыЧем эвристические алгоритмы отличаются от приближенных?
 
 Эвристические алгоритмы находят решения быстро, используя интуитивные правила, но не дают гарантий качества. Приближённые алгоритмы тоже решают задачи не идеально, но обеспечивают количественно измеримую точность относительно оптимального решения.

Примеры эвристических алгоритмов:  

Жадные алгоритмы  
Генетические алгоритмы  
Метод ближайшего соседа
Примеры приближённых алгоритмов:  

Алгоритм Кристофидеса для задачи коммивояжёра  
Аппроксимационный алгоритм для минимизации стоимости покрытий множеств  
Двухфазовые аппроксимирующие алгоритмы для задач планирования ресурсов
